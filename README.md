# AutoPress AI — Publication Automatique d'Articles

Plateforme de génération et publication automatique d'articles via l'API NVIDIA (build.nvidia.com).

## 🚀 Fonctionnalités

- ✨ **Génération d'articles** via l'API NVIDIA (modèles Llama, DeepSeek, Nemotron, etc.)
- 📝 **Interface blog** avec catégories, tags et filtres
- 📊 **Dashboard** avec statistiques de publication
- 🔑 **Configuration API** personnalisée (clé NVIDIA gratuite)
- 📱 **Responsive** design

## 🛠️ Configuration API NVIDIA

1. Va sur [build.nvidia.com](https://build.nvidia.com)
2. Crée un compte gratuit (Developer Program)
3. Génère une clé API (`nvapi-...`)
4. Saisis-la dans l'onglet **Paramètres** du site

**Gratuit** : 1 000 crédits, 40 req/min, pas de carte bancaire requise.
